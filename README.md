# 🚀 LearnSphere

LearnSphere is a full stack learning platform built using Java, Spring Boot, HTML, CSS, and JavaScript.

## ✨ Features
- User course browsing
- Course management
- Trainer and student modules
- Scalable backend integration
- Responsive frontend design

## 🛠 Tech Stack
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- MySQL
